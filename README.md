# 🌍 Wanderlust

A full-stack travel listing web application inspired by Airbnb, built using the MERN stack. Users can create, edit, and manage property listings, upload images, and leave reviews.

---

## 🚀 Features

- 🔐 User Authentication (Sign Up / Login / Logout)
- 🏡 Create, Edit & Delete Listings
- 📷 Image Upload using Cloudinary
- ⭐ Reviews & Ratings
- 📍 Interactive Map for Property Location
- 🔍 Server-side Validation
- 📱 Responsive Design

---

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/ShilpaV-de/wanderlust.git
```

Move into the project folder

```bash
cd wanderlust
```

Install dependencies

```bash
npm install
```

Start the server

```bash
npm start
```
